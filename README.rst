Installation
------------

greetpustat is distributed on `PyPI <https://pypi.org>`_ as a universal
wheel and is available on Linux/macOS and Windows and supports
Python 3.5+ and PyPy.

.. code-block:: bash

    $ pip install greetpustat

Usage
-----
.. code-block:: bash
    $ greetpustat --name"eric gerber"
